These examples focus on decisions you need to make. Every page links to compiling
repository examples when you need the complete package and generated output.

<CardGrid>
  <Card title="Optional and nullable" icon="open-book" href="/features/optionality/">
    Model an omitted property separately from a required property whose value may be null.
  </Card>
  <Card title="Enums" icon="list-format" href="/features/enums/">
    Generate string enums or expose meaningful names for integer constants.
  </Card>
  <Card title="Interfaces" icon="puzzle" href="/features/interfaces/">
    Register implementations and generate a discriminated `anyOf` schema.
  </Card>
  <Card title="Provider schemas" icon="setting" href="/features/providers/">
    Supply field schemas at runtime when static generation is not enough.
  </Card>
</CardGrid>
